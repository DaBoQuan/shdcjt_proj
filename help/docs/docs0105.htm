在线手册->知识文档管理->上传图片 后退 前进 

本系统提供了图片上传功能，上传的图片可以用作定义模板和网页。
系统将这些图片定义成如下两种类型：


1. 新闻页文档-这种图片将作为新闻页的特定位置上的LOGO使用


2. 新闻页背景-这种图片将作为新闻页的背景使用


向系统上传图片，选择文档模块，点击"维护、常用工具、图片上传、新建"，进入如下界面

 

下表是对相关字段属性的解释：

字段

解释信息

图片

说明

图片名称，此为必填项

类型

选择图片保存的类型

图片

选择需上传的图片文件

按"S-保存"按钮保存上述操作，如需传送其他图片请重复以上操作。
